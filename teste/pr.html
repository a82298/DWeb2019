    • ABSTRACT

    Neste teste, o objetivo seria praticar grande parte daquilo que aprendemos ao longo de semestre, sendo que terá um peso de 30% da nota final.

    Em primeiro lugar, no primeiro exercício, foi pedido que se construísse uma API de Dados (baseada num ficheiro JSON).

    A primeira tarefa foi transformar o XML em JSON, com todos os cuidados necessários e transformando tudo apenas numa lista. Depois foi realizado o comando mongoimport deste ficheiro JSON com o seguinte comando:

    mongoimport --db obras --collection obras --file {Caminho ate ao ficheiro JSON que se encontra na pasta do ex1 (obras.json)} --jsonArray

    O objetivo seria responder aos pedidos propostos pelo professor

    Por outro lado, no segundo exercício, o objetivo seria consumir uma API de dados fornecida e construir a respetiva interface WEB.

    Primeiro, foi realizado um teste sobre a API de dados e, através do Postman, respondeu-se às seguintes questões:

    -Quantas entidades estão catalogadas? R: 546

    -A que Tipologias pertence a Entidade "Autoridade para as Condições de Trabalho" (ent_ACT)? R: Adminstração Pública e Autoridades de supervisão e fiscalização

    -Em que processos a entidade "Autoridade Nacional de Segurança Rodoviária" (ent_ANSR) participa como Iniciador? R: Processo com o código 400.10.056

    -De que processos é dona a entidade "Colégio Militar" (ent_SEF)? R: Processo com o código 750.20.601

    Em seguida, montou-se o servidor a consumir a API da maneira pretendida.
